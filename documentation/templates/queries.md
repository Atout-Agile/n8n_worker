# Queries GraphQL

Liste des queries disponibles pour récupérer des données.

## Users
- user : récupère les informations d'un utilisateur
- users : liste les utilisateurs

## Tokens
- apiTokens : liste les tokens d'API actifs d'un utilisateur
- verifyToken : vérifie la validité d'un token et retourne ses informations (id, name, active, expiresAt, user)
